# Craft Attack 13 — TheJoeCraft Pack

This is an **unofficial community modpack** that bundles the mods, shaders, and required dependencies used by **TheJoeCraft** in **Craft Attack 13** into one easy-to-install package.

The goal of this pack is to make it simple for players to experience a setup that closely matches what is used in the series, without needing to install everything manually.

---

## What this pack includes

### Mods (Fabric)
This modpack is built entirely on **Fabric** and includes performance, visual, and utility mods such as:

- **Sodium** – Performance and rendering optimization  
- **Iris Shaders** – Shader support for Fabric  
- **Complementary Shaders** – The main shader used for visuals  
- **Replay Mod** – For recording smooth cinematic clips  
- **Freecam** – For free camera movement  
- **Zoom Mod** – Allows smooth and adjustable zooming  
- **UniFi** – Server compatibility and mod syncing  
- Required libraries and APIs (such as Cloth Config, Cloud Config API, and other dependencies) to ensure everything works together properly

> The exact mod list is managed automatically by Modrinth and will be downloaded when installing the pack.

---

## Visuals & Shaders

This pack comes configured to use **Complementary Shaders** via Iris, providing improved lighting, more natural skies, better reflections, and an overall modern visual style similar to Craft Attack 13. You can still change the shader in-game if you prefer a different look.

---

## Installation (Modrinth App — recommended)

1. Open the **Modrinth App**  
2. Search for **“Craft Attack 13 — TheJoeCraft Pack”**  
3. Click **Install**  
4. Launch the game from the Modrinth App
